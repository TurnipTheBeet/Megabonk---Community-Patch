# Megabonk Community Patch

A community-driven quality-of-life mod for Megabonk.

## Features

### Gameplay
- **Fast Fall** — hold Slide while airborne to fast fall
- **Tony McZooms** — Zap passive grants +0.25 projectiles per level
- **Golden Shield** — removes the reduced gold penalty when Kevin deals self-damage

### Item Tweaks
- **Item Cap Removal** — Anvil (max 2), Overpowered Lamp (max 3), Za Warudo (max 10) instead of default cap of 1
- **Spicy Meatball** — size capped at 16m; pool slot removed once cap is reached (accounts for size stat)
- **Grandma's Secret Tonic** — size capped at 16m; pool slot removed once cap is reached (accounts for size stat)

### Stat Blacklists
- Excludes junk/broken stats from Chaos, Gamble Tome, Dicehead passive, and Charge Shrines
- Configurable via `com.megabonk.mod.cfg`

### Community Leaderboard
- Submits your run stats to a shared leaderboard visible in-game
- Server endpoint configurable in `com.megabonk.mod.cfg`

## Controls

| Key | Action |
|-----|--------|
| F1  | Toggle mod menu |
| F2  | Toggle damage chart |
| Slide (airborne) | Fast fall |
