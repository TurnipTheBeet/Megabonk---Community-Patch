# MegaDuel

A quality-of-life mod for Megabonk.

## Features

- **Stat Blacklist** — exclude specific stats from Chaos, Gamble Tome, Dicehead, and Charge Shrines
- **Item Cap Removal** — remove stack caps on Anvil, Overpowered Lamp, and Za Warudo
- **Size Cap Removal** — remove size caps on Grandma's Secret Tonic and Spicy Meatball
- **Community Leaderboard** — submits your runs to a shared leaderboard visible in-game

## Usage

Press **F1** in-game to open the mod menu.
